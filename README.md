# SHARDS Tokens

A token vision and light preset manager for the SHARDS Dock. Apply vision modes and light sources to selected tokens in one click — with the ability to restore their original state at any time.

> **Requires:** [SHARDS Core](https://github.com/dev-shua/shards-core)

---

## Features

- **Vision presets** — apply a vision mode (darkvision, blindness, etc.) to the selected token instantly
- **Light presets** — apply a light source (torch, no light, etc.) to the selected token instantly
- **Restore** — the token's original state is saved automatically before any preset is applied. Revert it at any time with a single click
- **Custom presets** — create your own vision or light presets with full control over range, angle, color, animation, and more
- **Multi-token support** — works on all currently selected tokens at once

---

## Installation

In Foundry VTT, go to **Add-on Modules → Install Module** and paste the manifest URL:

```
https://raw.githubusercontent.com/dev-shua/shards-tokens/main/public/module.json
```

Make sure **SHARDS Core** is installed and enabled first.
